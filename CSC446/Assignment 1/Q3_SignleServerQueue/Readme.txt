Please compile all 4 java files first before running them

To compile:
javac Event.java EventList.java Queue.java Sim.java

To run:
java Sim xxxx
**xxxx is the initial seed number

To change the number of customers in the simulation, edit the "TotalCustomers" variable at line 22
in Sim.java. Proper implementation of mean service time and interarrival time has been added
